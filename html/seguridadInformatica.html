ABOUT ME

La ciberseguridad es la practica de defender computadoras, los servidores,los dispositivos moviles, los sistemas electronicos, las redes y los datos de ataques maliciosos.Tambien se conoce como seguridad de tecnologia de la informacion o seguridad de la informacion electronica. El termino se aplica en diferentes contextos, desde los negocios hasta la informatica movil, y puede dividirse en algunas categorias comunes

La seguridad de red es la practica de proteger una red informatica de los intrusos, ya sea atacantes dirigidos o malware oportunistas.

La seguridad de las aplicaciones se enfoca en mantener el software y los dispositivos libres de amenazas. Una aplicacion afectadas podria acceder a los datos que esta destinada a proteger. La seguridad eficaz comienza en la etapa de diseño, mucho antes de la implementacion de un programa o dispositivo.

La seguridad de la informacion protege la integridad y la privacidad de los datos, tanto en el almacenamiento como en el transito.

La seguridad operativa incluye los procesos y decisiones para manejar y proteger los recursos de datos. Los permisos que tienen los usuarios para acceder una red y los procedimiento que determinan como y donde pueden almacenarseo o compartirse los datos se incluyen en esta categoria.

MALWARE

"Malware" se refiere al software malicioso. Ya que es una las ciberamenazas mas comunes, el malware es software que un cibercriminal o un hacker ha creado para interrumpir o dañar el equipo de un usuario legitimo. Con frecuencia propagado a traves de un archivo adjunto de correo electronico no solicitado o de una descarga de apariencia legitima, el malware puede ser utilzado por los ciberdelincuente para ganar dinero o para realizar ciberataques con fines politicos.

Hay diferentes tipos de malware, entre los que se incluyen los siguientes:

*Virus: un programa capaz de reproducirse, que se inscrusta un archivo limpio , y se extiende por todo le sistema informatico e infecta los archivos con codigo malicioso.

*Troyanos: un tipo de malware que se disfraza como SOFTWARE legitimo. Los cibercriminales engañan a los usuarios para que carguen troyanos a sus computadoras, donde causan daños o recopilan datos.

*Spyware: un programa que registra en secreto lo que hace un usuario para que los cibercriminaales puedan hacer uso de esta informacion. Por ejemploço, el spyware podria capturar los detalles de las tarjetas de credito.

*Adware: Software de publicidad que puede utilizarse para difundir malware.

*Botnets: Redes de computadoras con infeccion de malware que los cibercriminales utilizan para realizar tareas en linea sin el permiso del usuario.


Inyeccion de codigo SQL

Una inyeccion de codigo SQL es un tipo de ciberataque utilizado para tomar el control y robar datos de una base de datos. Los cibercriminales aprovechan las vulnerabilidades de las aplicaciones basadas en datos para insertar codigo malicioso en una base de datos mediante una instruccion SQL maliciosa. Esto le brinda acceso a la informacion confidencial contenido a la base de datos.

Phishing

El phishing es cuando los cibercriminales atacan a sus victimas con correos electronicos que parecen ser de una empresa legitima que solicita informacion confidencial. Los ataques de phishing se utiliza a menudo para inducir a que las personas entreguen sus datos de tarjetas credito y otra informacion personal.


Ataque de tipo Man-in-the-middle

Un ataue de tipo Man-in-the-middle es un tipo de ciberamenaza en la que un cibercriminal intercepta la comunicacion entre dos individuos para robar datos. Por ejemplo, en una red WI-FI segura, un atacante podria interceptar los datos que se transmiten desde el dispositivos de la victima y la red.

Ataque de denegacion de servicios

Un ataque de denegacion de servicios es cuando los cibercriminales impiden que un sistema informatico satisfaga solicitudes legitimas sobrecargando las redes y los servidores con trafico. Esto hace que el sistema sea inutilizable e impide que una organizacion realice funciones vitales.
